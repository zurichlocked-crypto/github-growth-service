Security Policy

Supported Versions
This repository primarily contains documentation and educational resources. Security updates will be applied as needed to maintain the integrity of the project.

Reporting a Security Issue
If you believe you have discovered a security issue related to this repository, please do not disclose it publicly.
Instead, contact the repository maintainer with a detailed description of the issue. Reports will be reviewed promptly, and appropriate action will be taken if necessary.

Thank you for helping keep this project safe and reliable.
